# Como executar o projeto
Pré-requisitos: node/yarn

```bash
# clonar repositório
git clone https://github.com/gabrielssantana/nlw.git

# entrar na pasta do projeto front end web
cd nlw

# instalar dependências
yarn install

# executar o projeto
yarn dev

#Acessar http://localhost:5500
```
